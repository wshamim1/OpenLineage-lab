# Data Directory

This directory contains sample data files generated by the examples.

## Contents

When you run the Python lineage example, it will create:
- `customers.csv` - Sample input data (5 customer records)
- `customers_filtered.csv` - Filtered output data (customers over 30)

## Note

These files are automatically generated and are ignored by git (see `.gitignore`).

To regenerate the data, run:
```bash
cd ../examples/python
python3 python-lineage-example.py
